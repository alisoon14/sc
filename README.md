ветка для показа работы за первую практику
Базовый бот + БД + основные команды

Цель: поднять рабочий Telegram-бот с доступом к базе и простыми командами.

Включает:
- main.py
- config.py
- helpers.py
- db_operations.py
- command_handlers.py
- shift_operations.py
- keyboards.py (если нужны кнопки)

Функционал:
- загрузка .env
- подключение к MySQL
- команда /start
- команда /refresh
- создание напоминания/основного запроса
- вывод информации пользователю
- проверка активности пользователя в Employees

Это основа, на которой строится весь остальной функционал.
